Công Cụ Chuyển Excel → data.json

Dùng để cập nhật dữ liệu thí sinh mỗi kỳ thi — không cần cài phần mềm

Hướng dẫn sử dụng

1

Chuẩn bị file Excel với các cột: SBD, Họ tên, Ngày sinh, Lớp, SĐT, Phòng thi

2

Điền thứ tự cột tương ứng vào bảng bên dưới (cột A = 1, cột B = 2, cột C = 3...)

3

Kéo thả hoặc chọn file Excel → xem trước dữ liệu → tải về data.json

4

Upload file data.json mới lên server, ghi đè file cũ → website tự cập nhật
